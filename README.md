# Dumpr

Static landing pages for **Dumpr** — an iOS app that turns one rambling
brain dump into clean items in your reminders and calendar.

Hosted via GitHub Pages at <https://dumpr.agnticstudio.com>.

## What's served

- `/` — marketing page
- `/dump` — share-dump receiver fallback (renders for visitors without
  the iOS app installed; deep-links straight into the app for those
  who do)
- `/.well-known/apple-app-site-association` — Apple Universal Link
  verification
